How to generate test data

Run the test script to generate a JSON file with suggestions:

# Set your OpenAI API key
export OPENAI_API_KEY=sk-...

# Generate suggestions
python test_suggestions.py

# This creates: test_suggestions_output.json

Then click "Load JSON file" above and select test_suggestions_output.json.

What to expect in editor mode

  • Yellow highlights: Pending suggestions (hover to see details)
  • Green highlights: Accepted suggestions (with strikethrough)
  • Dimmed text: Ignored suggestions
  • Toolbar: Shows suggestion counts and "Copy result" button
  • Popup: Hover over highlighted text to see the suggestion, accept or ignore it
